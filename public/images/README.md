e# Imagens

Adicione aqui as imagens do projeto. Dimensões recomendadas:

- `hero-dog.jpg` — 1200x900px (foto do hero, cão em ambiente natural)
- `about-dogs.jpg` — 800x1000px (foto dos cães Zoro e Kaya)
- `petisco-maca-canela.jpg` — 600x600px (produto: petisco maçã e canela)
- `chifre-bovino.jpg` — 600x600px (produto: chifre real bovino)
- `mix-proteinas.jpg` — 600x600px (produto: mix proteínas nobres)
- `og-image.jpg` — 1200x630px (imagem Open Graph para redes sociais)
- `logo.png` — 200x60px (logo para schema JSON-LD)

Use imagens em formato JPG/WebP. O Next.js converte automaticamente para AVIF/WebP no build.
